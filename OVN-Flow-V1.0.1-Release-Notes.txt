OVN-Flow Release Notes

OVN-Flow is an Open-Source lightweight, minimal dependency VPN client with a clear goal. To simplify OpenVPN Connection Management. Offering Multi-Config Support, Multi-Step Connection Verification and a clean, simple interface. We aim to provide regular updates to our project. 

Version 1.0.1:

1 - Added a backup connection to external connectivity testing utilizing Google services (8.8.8.8) ping if Cloudflare (1.1.1.1) fails.

2 - Improvements to User Interface (Configuration Menu & Main Menu).

3 - Modularised all connection processes to separate functions. Allowing for effective script maintenance.

4 - Updated comments and categorized functions 

5 - Removed Figlet Banner art dependency.
